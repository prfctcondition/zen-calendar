# Zen Calendar

A lightweight desktop calendar widget that sits on your screen and shows your Google Calendar events.

## Features

- Monthly calendar view and daily event list
- Google Calendar integration (read-only)
- Sound notifications 5 minutes before events and at event start
- Always-on-top mode
- Black & white minimalist theme
- Drag anywhere to move

## Download

[Download the latest installer](https://github.com/prfctcondition/zen-calendar/releases/latest)

## How to use

1. Install the app
2. Sign in with your Google account
3. View your upcoming events
